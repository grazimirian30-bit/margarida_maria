Faça tudo pelo Painel Administrativo: Execute o projeto: python app.py Abra no navegador: http://127.0.0.1:5000 Clique em: Painel Administrativo Entre com: Usuário: grazi Senha: margarida2026 Clique em: + Novo produto Preencha: nome; categoria; preço; estoque; cor; tamanho; material; selo; descrição. Na parte Imagens do produto, clique para escolher as fotos. Você pode selecionar várias de uma vez usando: Ctrl + clique A primeira imagem escolhida será a principal. As demais aparecem na galeria quando o cliente abre os detalhes. Depois clique em: Cadastrar produto As imagens serão salvas automaticamente em: static/img/produtos E o produto será gravado no banco: catalogo.db Para trocar ou acrescentar imagens depois, entre no painel, clique em Editar, marque as imagens antigas que deseja remover e selecione novas fotos. Não precisa abrir código.
