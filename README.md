# Catálogo Margarida Maria — V4 Final Completa

Esta versão reúne tudo do v3 com todas as funções novas do v4.

## Conteúdo incluído

- Logo e identidade visual.
- Imagens e produtos da versão v3.
- 24 produtos já cadastrados.
- Galeria com várias imagens.
- Zoom nas fotos.
- Pesquisa e filtros.
- Cadastro de clientes.
- Avisos que desaparecem automaticamente.
- Banco de dados preservado.
- Painel Administrativo.
- Cadastro de produtos pelo navegador.
- Upload de várias imagens.
- Edição e exclusão de produtos.
- Controle de estoque.
- Produtos em destaque.
- Visualização de clientes cadastrados.

## Como executar

Abra a pasta no VS Code e rode:

```bash
python -m pip install -r requirements.txt
python app.py
```

Depois abra:

```text
http://127.0.0.1:5000
```
